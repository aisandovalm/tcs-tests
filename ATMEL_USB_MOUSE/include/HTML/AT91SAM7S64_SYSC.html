OffsetNameDescription
0SYSC_PSTDBY
AT91C_SYSC_PSTDBY		Voltage Regulator Power Mode
0 = Voltage Regulator is in Normal Power Mode.
1 = Voltage Regulator is in Standby Power Mode (low power mode).
